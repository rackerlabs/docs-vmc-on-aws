==================
Spheres of support
==================

.. toctree::
   :maxdepth: 2

   sec-spheres-supported-vm.rst
   sec-spheres-unsupported-vm.rst

Spheres of support and support requests for Dedicated VMware vCloud are
similar to the current Server Virtualization product. With Dedicated
VMware vCloud, you can create supported and unsupported VMs. The
Rackspace virtualization team maintains the Dedicated VMware vCloud
configuration and services to ensure uptime.
