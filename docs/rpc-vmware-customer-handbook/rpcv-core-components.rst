==========================
RPC-VMware core components
==========================

RPC-VMware is a dedicated, per-customer instance of the VMware SDDC stack.
As standard, RPC-VMware includes an ESXi hypervisor running on the following:

* Dedicated hardware
* VMware vSphere
* VMware vSAN software-defined storage platform
* VMware NSX software-defined networking platform
* VMware vRealize Operations

Other services, including Managed Backup (MBU),
are available as optional add-ons to RPC-VMware. Further details on the
components included as standard are found below:
