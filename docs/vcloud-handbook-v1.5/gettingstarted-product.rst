===================
Product description
===================

Dedicated vCloud provides a single-tenant hosted private cloud designed
with server, storage, and networking hardware from vendors like Dell,
EMC, and Cisco. Dedicated vCloud is managed 24x7x365 by Rackspace VMware
Certified Professionals. Dedicated VMware vCloud provides a cloud
management platform to build upon, while maintaining control through the
vCloud Director web portal and the vCloud API.
